# Sonho de Infância — Landing Page

Landing page estática feita com HTML, CSS e JavaScript puro. Não usa Node.js, npm, build, módulos ES6 ou servidor de aplicação.

## Executar localmente

1. Abra a pasta no VS Code.
2. Clique com o botão direito em `index.html`.
3. Escolha **Open with Live Server**.

Também pode abrir o `index.html` diretamente no navegador.

## Publicar no GitHub Pages

1. Envie todo o conteúdo desta pasta para a raiz do repositório.
2. No GitHub, acesse **Settings > Pages**.
3. Em **Build and deployment**, escolha **Deploy from a branch**.
4. Selecione a branch `main` e a pasta `/ (root)`.
5. Salve.

## Estrutura

- `index.html`: página principal.
- `assets/css/`: estilos.
- `assets/js/app.js`: menu, preloader, animações, galeria, FAQ e interações.
- `assets/img/`: imagens oficiais e mockups fornecidos.
